Here are descriptions of some of the code we've worked on in class!

American Flag

Task: Create an american flag using bootstrap rows/columns

For this assignment, we were told to recreate an American flag by using bootstrap. Lots of containers, rows, columns and all that good stuff!


Operations Calculator

Task: Create a two number calculator

For this assignment, we were told to create a two number calculator. The page prompted a selection of two numbers, and then prompted an operation (add, multiply, subtract, divide).

The calculator would then solve the equation based on the numbers and operation picked - cool stuff!


Rock, Paper, Scissors

Task: Create a game of rock, paper, scissors by inputting decisions

For this assignment, we were to create a working game of rock, paper, scissors that looped 10 times. The user would input any choice of the 3, then the computer would do the same.

The computer would tell you if you won, lost, or tied. It would repeat 10 times and then give a final score at the end. This one was hard!
